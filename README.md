## :rocket: Quick Start

### 1. Install gatsby-cli

```sh
$ npm install -g gatsby-cli
```

### 2. Create Project

```sh
$ gatsby new my-blog-starter https://github.com/kimnagui/gatsby-starter-nagui
```

### 3. Start Project

```sh
$ cd my-blog-starter
$ gatsbyjs develop
# open http://localhost:8000
```

## :star: Features

-   Tags & Categorys.
-   Pagination.

-   Atomic Design Components.
